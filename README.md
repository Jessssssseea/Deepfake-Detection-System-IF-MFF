# AI-FaceForgery-Detection-IsolationForest
## 基于多维特征融合与孤立森林的AI深度伪造换脸检测系统研究
基于**多维特征融合**与**孤立森林**(Isolation Forest)的AI换脸（深度伪造）检测系统，实现无监督的人脸伪造图像/视频检测，无需海量标注伪造样本，仅通过真实人脸数据训练，轻量高效、可解释性强。

## 项目简介
本项目针对AI换脸技术滥用带来的虚假信息传播、电信诈骗等问题，构建了一套无监督异常检测模型：
1. 提取人脸**几何特征、光照一致性、纹理细节、颜色分布、频域特征、眼球反射一致性**6类核心特征，拼接为54维特征向量；
2. 基于孤立森林算法学习真实人脸的特征分布，通过异常得分判别人脸真伪；
3. 支持**静态图像检测**与**视频帧检测**，提供图形化界面（GUI），适配低算力场景；
4. 在近60万张样本的测试集上实现**AUC=0.8265**、整体准确率78.92%，真实人脸检测准确率87%。

### 核心优势
- 📊 **可解释性强**：每个特征均有明确物理意义，可精准定位伪造破绽；
- ⚡ **轻量高效**：CPU可完成特征提取与推理，单张图片检测速度快，部署成本低；
- 🌐 **泛化性好**：基于统计分布检测，对未见过的换脸算法具有鲁棒性；
- 🔧 **无监督训练**：仅需真实人脸样本，无需标注伪造数据，训练成本低；
- 📈 **多维特征融合**：6大维度全方位捕捉换脸图像的异常特征，突破单一特征的局限性。

## 快速开始
### 1. 数据准备
将真实人脸图像和伪造人脸图像分别放入如下目录（需自行创建）：
```
data/
├── real/  # 真实人脸图像（JPG/PNG/BMP）
└── fake/  # 伪造人脸图像（JPG/PNG/BMP）
```
### 2. 模型训练
运行训练脚本，自动完成特征提取、模型训练、阈值确定，并保存模型/标准化器/阈值：
```bash
python isolationforest_train.py
```
训练完成后会生成：
- `isolation_forest_model.pkl`：训练好的孤立森林模型
- `scaler.pkl`：特征标准化器
- `threshold.txt`：检测阈值（验证集95%分位数）
- `training_log.txt`：训练日志
- `roc_curve.png`：ROC曲线可视化图

### 3. 模型测试
运行测试脚本，自动计算检测指标并生成测试报告：
```bash
python test_model.py
```
测试完成后会生成：
- `report.txt`：详细测试报告（准确率、AUC、混淆矩阵等）
- 各类可视化图（PR曲线、异常得分分布、混淆矩阵等）

### 4. 图形界面检测（图片/视频）
运行GUI脚本，通过可视化界面完成图片/视频换脸检测：
```bash
python Qmain.py
```
- **图片检测**：点击「浏览」选择图片，点击「检测」即可得到真伪结果+异常得分；
- **视频检测**：选择视频文件，设置抽帧策略，自动逐帧检测并统计结果。

## 核心算法
### 1. 多维特征提取（54维）
| 特征类型         | 核心计算内容                     | 维度 |
|------------------|----------------------------------|------|
| 几何特征         | 对称点距离差、眼宽/脸宽、嘴宽/脸宽 | 6    |
| 光照一致性特征   | 左右脸灰度均值差                 | 1    |
| 纹理特征         | LBP直方图（前30维）+高频噪声能量 | 31   |
| 颜色分布特征     | 肤色Cr直方图（前10维）+唇色/肤色差 | 11   |
| 频域特征         | 低频/高频能量占比                | 2    |
| 眼球反射特征     | 左右眼高光区域比例差             | 1    |
| **总计**         |                                  | **54**|

### 2. 孤立森林异常检测
- 构建100棵孤立树，随机选择特征和分割值划分样本，异常点（伪造人脸）更易被孤立，路径长度更短；
- 异常得分归一化至(0,1)，得分越接近1越可能为伪造；
- 取验证集真实样本的95%分位数作为检测阈值，保证95%的真实样本被正确分类。

### 3. 判别规则
$$
label(x) = 
\begin{cases} 
\text{Real}, & S(x) \leq \tau \\
\text{Fake}, & S(x) > \tau 
\end{cases}
$$

其中，
$$S(x)$$
为异常得分，
$$\tau$$
为检测阈值。

## 模型拓展
1. **视频检测**：基于静态图像检测模型，通过抽帧策略逐帧检测，综合时序信息提升鲁棒性；
2. **多模型融合**：可集成Xception、ViT等有监督深度学习模型，构建综合检测平台；
3. **特征优化**：引入深度特征（如CNN提取的视觉特征），弥补手工特征的表达能力不足；
4. **场景适配**：针对社交媒体、取证、新闻媒体等场景优化检测策略，提升实际应用效果。

## 项目不足与改进方向
### 现有不足
1. 手工特征对**遮挡、大角度人脸**检测效果不佳，对高质量合成换脸图像识别易出错；
2. 特征设计依赖人工知识，可能遗漏隐蔽的换脸破绽；
3. 孤立森林对**高维、多模态数据**的适应性有限，难以刻画真实人脸的复杂特征分布。

### 改进方向
1. 融合**手工特征+深度特征**，提升特征的表达能力和泛化性；
2. 引入**时序一致性约束**（如光流、人脸关键点运动轨迹），优化视频检测效果；
3. 改进孤立森林算法，或融合One-Class SVM、自编码器等模型，提升异常检测能力；
4. 增加**数据增强**，覆盖更多光照、姿态、遮挡场景，提升模型鲁棒性。

## 应用场景
